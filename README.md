# ☕ Salve Shop

## Overview
**Salve Shop** is a web application designed for a local coffee shop. Currently in its initial front-end development phase, this project aims to showcase the café's brand while transitioning into a fully functional digital storefront for coffee lovers. 

## ✨ Current Features
* **Welcoming User Interface:** A warm design that reflects the cozy vibe of a coffee shop.
* **Static Front-End Structure:** Built with core web technologies for instant loading.
* **Custom Styling:** Fully styled using custom CSS to establish the café's aesthetic.

## 🚧 Roadmap & Upcoming Features
This project is continuously evolving. The current repository contains the foundational HTML/CSS, but the following features are actively in the pipeline for future releases:

* [ ] **User Account System:** Registration and Sign-Up / Login functionalities for customers.
* [ ] **Dynamic Digital Menu:** An interactive menu for users to browse coffee varieties, pastries, and prices.
* [ ] **Checkout & Receipts:** A cart system that processes orders and generates digital billing receipts.
* [ ] **Order Tracking:** A live status tracker so customers know when their coffee is being prepared or is ready for pickup/delivery.
* [ ] **Backend Integration:** Future implementation of JavaScript and database services (like Firebase or MySQL) to power the data and mechanics, plus many more upcoming features!

## 💻 Tech Stack (Current)
* **HTML5:** For the semantic structure of the website (`index.html`).
* **CSS3:** For styling, layout, and visual presentation (`style.css`).
